# Library-of-Vasca
Submission for DigiTech 2025 semester 2

Requires SQLite studio, python and Pycharm, and a web browser such as google or microsoft edge to run, along with a compatible operating system such as windows 11.

Python - https://www.python.org/
Pycharm - https://www.jetbrains.com/pycharm/
SQLite studio - https://sqlitestudio.pl/

To install download flask_recipes.zip and extract all, open Pycharm and drag the unzipped folder somewhere into the pythonproject folder (project window). AFter that, right click app.py and select Run 'app'.
